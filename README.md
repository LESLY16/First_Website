# Lesly Dsouza | Portfolio Website

A modern, professional portfolio website built with React and Framer Motion, showcasing Lesly Dsouza's professional profile as a Resident Canadian Software Engineer.

## 🚀 Features

- **Smooth Animations** – Framer Motion powered entrance, scroll, and hover animations
- **Responsive Design** – Mobile-first layout with hamburger navigation
- **Interactive UI** – Animated skill cards, timeline experience, contact links
- **Performance** – Optimized 60fps animations, lazy scroll triggers via `useInView`

## 🛠️ Tech Stack

- **React 18** with hooks (useState, useEffect, useRef)
- **Framer Motion 10** for all animations
- **CSS Variables** for consistent theming
- **Google Fonts** – Inter & Poppins

## 📦 Getting Started

### Prerequisites

- Node.js ≥ 16
- npm ≥ 8

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000).

### Build for Production

```bash
npm run build
```

The optimized build will be in the `build/` directory.

## 📁 Project Structure

```

    ├── Experience.js        # Work experience timeline

```
